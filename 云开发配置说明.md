# 微信小程序云开发配置说明

## 概述

为了防止用户清理微信缓存导致数据丢失，我们集成了**微信云开发**功能，将用户数据自动备份到云端。

## 数据保护层级

现在你的小程序有**四层数据保护**：

1. **本地主存储** - `wx.setStorageSync`
2. **本地备份存储** - 自动创建备份副本
3. **本地历史记录** - 保留最近30个版本
4. **云端存储** - 微信云开发数据库

## 配置步骤

### 1. 开通微信云开发

1. 登录[微信公众平台](https://mp.weixin.qq.com/)
2. 进入你的小程序管理后台
3. 点击左侧菜单 "云开发"
4. 点击 "开通" 按钮
5. 选择 "免费版" 即可（足够个人使用）

### 2. 创建数据库集合

1. 进入云开发控制台
2. 点击 "数据库"
3. 点击 "添加集合"
4. 输入集合名称：`user_data`
5. 点击 "确定"

### 3. 配置数据库权限

1. 在数据库页面，点击 `user_data` 集合
2. 点击 "权限设置"
3. 选择 "所有用户可读，仅创建者可读写"
4. 点击 "保存"

### 4. 修改 app.json

确保你的 `app.json` 包含云开发配置：

```json
{
  "cloud": true,
  "pages": [
    "pages/index/index",
    "pages/tasks/tasks",
    "pages/stats/stats",
    "pages/login/login"
  ]
}
```

## 数据同步机制

### 自动同步

- **启动时**：自动从云端恢复数据（如果本地为空）
- **数据变更时**：自动同步到云端
- **定期同步**：每5分钟自动同步一次

### 手动同步

在任务管理页面，可以手动触发同步（如果需要添加同步按钮）。

## 数据恢复流程

当用户打开小程序时：

```
1. 检查本地存储
   ↓ 如果为空
2. 尝试从云端恢复
   ↓ 如果云端有数据
3. 恢复数据到本地
   ↓ 如果云端也没有
4. 初始化默认数据
```

## 用户清理缓存后的恢复

即使用户清理了微信缓存：

1. 用户重新打开小程序
2. 系统自动检测到本地数据为空
3. 自动从云端下载数据
4. 用户的数据完整恢复

**注意**：用户需要使用同一台设备或登录同一微信账号。

## 免费额度说明

微信云开发免费版包含：

- 数据库容量：2GB
- 读操作：5万次/天
- 写操作：3万次/天

对于个人使用的小程序完全足够。

## 常见问题

### Q: 用户没有网络怎么办？
A: 数据会先保存到本地，有网络时自动同步到云端。

### Q: 云端数据会丢失吗？
A: 微信云开发的数据非常安全，不会丢失。

### Q: 可以导出云端数据吗？
A: 可以在云开发控制台导出数据备份。

### Q: 如何查看云端数据？
A: 登录微信公众平台 → 云开发 → 数据库 → user_data 集合

## 技术实现

主要代码文件：

- `utils/cloudSync.js` - 云端同步工具
- `app.js` - 初始化云开发和数据恢复
- `pages/tasks/tasks.js` - 任务数据变更时同步到云端
