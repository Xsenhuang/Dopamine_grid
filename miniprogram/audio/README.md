# 音频文件说明

此目录用于存放本地音频文件。

## 需要的音频文件

1. **complete.mp3** - 任务完成时的音效
   - 建议时长：1-2秒
   - 音量：适中，清脆悦耳
   - 推荐类型：提示音、成功音

2. **bingo.mp3** - BINGO连线时的庆祝音效
   - 建议时长：2-3秒
   - 音量：比complete.mp3稍大
   - 推荐类型：庆祝音、胜利音

## 获取音频文件的途径

### 方案1：从免费音效网站下载
- [Pixabay](https://pixabay.com/sound-effects/) - 免费商用
- [FreeSound](https://freesound.org/) - 需要注册
- [Zapsplat](https://www.zapsplat.com/) - 免费注册

### 方案2：使用微信小程序自带音效
如果不想添加音频文件，可以修改代码使用微信小程序的振动反馈代替音效。

### 方案3：使用Base64编码的短音效
可以将短音效转换为Base64编码直接嵌入代码中（适合非常短的音效）。

## 音频格式要求

- 格式：MP3
- 采样率：44.1kHz 或 48kHz
- 比特率：128kbps 或以上
- 文件大小：建议单个文件不超过 500KB

## 注意事项

1. 音频文件会打包进小程序，影响包体积
2. 建议音频文件总大小控制在 1MB 以内
3. 如果不需要音效功能，可以将音频文件留空，代码会自动静默处理
